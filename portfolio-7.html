Swift Data Aggregator

During my time at Nascent Digital as a Software Developer Co-op, I worked on the iOS and Android versions of a health sharing and messaging platform for a corporate client. One major issue we ran into during development was the time it took to manually input large amounts of health data to test the graphing functionality of the main project. I was given the opportunity to take lead on developing a secondary iOS application whose purpose is to aggregate large amounts of data into Apple's default "Health" application, simplifying the testing process. This project involved the use of:

  • Writing to and reading from the HealthKit API
  • UIKit animations (BONUS: The shark dances when you save data!)
  • Generating random data points based on user input

This project took about 2 weeks to develop (in Swift) from wireframe to final product. It drastically increased the debugging and testing efficiency of the main application (about 3X more efficient!), and eventually made its way into the Nascent toolbox to be used by other employees for different projects!

Unfortunately, since this was an internal project, the code is not publicly available
